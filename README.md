# Billy The Data Bot
## Automating csv data generation


## Features

- Read input json file
- Generates data based on json data and exports output file

## Installation

Billy-the-data-bot requires 'faker' library to run.


```sh
pip install faker
```

## How to run
```sh
python generate.py <input file>
```
